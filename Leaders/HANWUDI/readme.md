汉武大帝
领袖能力-孝武皇帝：
击杀单位获得+4大将军点数，解锁学院区域同时解锁太学（代替大学）。

特色单位-武刚车（代替重型战车）：
武刚车是中国古代的一种兵车，是防御骑兵冲锋的利器，用于陆上战斗。有至少一个相邻武刚车单位时，+10战斗力。

特色建筑-太学（代替大学）：
太学是中国古代的国立最高学府，是世界上最早的大学，比西方早了1000多年。代替大学，汉武帝在解锁学院区域的同时也可以解锁太学（属性与大学相同）。

历史议程-大汉天威：
犯我大汉天威者，虽远必诛！宣布正式战争后十回合内+5战斗力，+2移动力。
